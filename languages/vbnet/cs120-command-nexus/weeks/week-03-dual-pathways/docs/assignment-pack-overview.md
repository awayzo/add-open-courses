# Week 3 VB.NET Project Pack

This folder contains the Week 3 project deliverables requested for the dual-path assignment.

## Included Projects

### 1. `Week3DualPathwaysMaster`
Completed reference project with both pathways finished:

- Frex // Agra Route Recall
- MAS // Asteroid Response Alert

Use this as the instructor check, answer key, or tutorial reference build.

### 2. `AgraRouteRecallTemplate`
Starter template for the Frex tutorial.

What is already included:

- the agra map image
- title and output labels
- a guided build area
- beginner-friendly procedure and function stubs

What the learner still does:

- add four buttons
- name them North, East, South, and West
- write the button click logic
- connect input to output
- repair 3 intentional startup fixes before the app will run

### 3. `AsteroidResponseAlertTemplate`
Starter template for the MAS tutorial.

What is already included:

- the asteroid field image
- title and output labels
- hotspot guide boxes
- beginner-friendly procedure and function stubs

What the learner still does:

- create hotspot buttons
- name and style them
- write the conditional response logic
- connect input to output
- repair 3 intentional startup fixes before the app will run

## Week 4 Concepts Included

All three projects intentionally carry Week 4 structure too:

- procedures
- functions
- simple encapsulation

The code uses named procedures and functions so the learner can see that UI work
and back-end logic do not need to live inside one giant click event.
