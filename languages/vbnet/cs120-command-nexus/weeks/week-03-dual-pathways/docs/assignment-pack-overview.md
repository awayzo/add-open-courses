# Week 3 VB.NET Project Pack

This pack contains the public project files for the **Week 3 Dual Pathways**
assignment.

## Included Projects

### `Week3DualPathwaysMaster`

Completed reference project with both pathways finished:

- Frex // Agra Route Recall
- MAS // Asteroid Response Alert

Use this when you want to inspect a working finished version of the assignment.

### `AgraRouteRecallTemplate`

Starter template for the Frex pathway.

This project gives learners a guided starting point while still leaving real
work for them to complete. It includes:

- the agra map image
- title and output labels
- a guided build area
- beginner-friendly procedure and function structure

Learners still need to:

- add four direction buttons
- name them North, East, South, and West
- write the button click logic
- connect input to output
- repair 3 intentional startup fixes before the app will run

### `AsteroidResponseAlertTemplate`

Starter template for the MAS pathway.

This project also begins with guided structure and leaves the learner with real
UI and logic work to complete. It includes:

- the asteroid field image
- title and output labels
- hotspot guide boxes
- beginner-friendly procedure and function structure

Learners still need to:

- create hotspot buttons
- name and style them
- write the conditional response logic
- connect input to output
- repair 3 intentional startup fixes before the app will run

## Learning Direction

These projects are designed to introduce or reinforce:

- input and output flow
- interface behavior
- conditional logic
- simple procedures
- simple functions
- early organization inside a form-based app

The guided templates are meant to help learners move forward even when every
piece of the code is not yet fully familiar.
