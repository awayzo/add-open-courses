# Public vs Private Scope

This repository is meant to help build an open teaching reputation without
giving away every internal course asset.

## Recommended Model

Use a **split system**:

### Public Repo

Keep these here:

- learner starter templates
- selected master reference builds
- assignment packs you want to showcase
- reusable docs
- open assets you are comfortable sharing
- public releases for learner downloads

### Private Course Operations Repo or Local Workspace

Keep these out of the public repo:

- gradebook-connected material
- answer keys you do not want publicly visible
- unreleased future weeks
- private media drafts
- institution-specific operations docs
- anything tied to paid, internal, or still-experimental delivery

## Best Practical Rule

If the file helps your reputation and still works well as a public teaching
artifact, publish it.

If the file gives away internal delivery, grading, or future release advantage,
keep it private.

## Good First Public Scope

Publish:

- Week 2 starter pack
- Week 3 dual pathways pack
- selected Week 4 structure pack

Keep private for now:

- full semester delivery internals
- all production media source files
- internal assessment and grading scaffolds
