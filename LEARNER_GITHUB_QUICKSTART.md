# Learner GitHub Quick Start

If this is your first time using GitHub, that is completely fine.

You do **not** need to know everything about GitHub to start this course. For
Week 3, you only need enough understanding to find the right project pack and
download it correctly.

## What GitHub Is

GitHub is a place where people store, organize, and share code projects.

Developers use it to:

- keep project files together
- track changes over time
- share starter code
- publish open-source work
- collaborate with other people

You will learn more about GitHub later. For now, just think of it as the place
where your starter templates live.

## Why Learning GitHub Matters

Even at a beginner level, GitHub is worth learning because it helps you:

- find real project files in an organized way
- download starter code safely
- follow updates over time
- build a portfolio later
- understand how modern software projects are shared

This week, GitHub is not the main subject. It is just part of becoming more
comfortable working with real project materials.

## What You Need To Do This Week

For Week 3, your goal is simple:

1. open the correct GitHub template folder
2. download the project
3. open it in Visual Studio
4. begin the guided build

## Step-By-Step: How To Download A Template From GitHub

### Option 1 // Download the whole repository as a ZIP

Use this if you want the simplest path.

1. Open the repo homepage:
   - [ADD Open Courses](https://github.com/awayzo/add-open-courses)
2. Click the green **Code** button.
3. Click **Download ZIP**.
4. Save the ZIP file somewhere easy to find.
5. Right-click the ZIP and choose **Extract All**.
6. Open the extracted folder.
7. Go to:
   - `languages/vbnet/cs120-command-nexus/weeks/week-03-dual-pathways/templates/`
8. Open the template you need:
   - `AgraRouteRecallTemplate`
   - or `AsteroidResponseAlertTemplate`

### Option 2 // Browse directly to the Week 3 template folder first

Use this if you want to understand the repo layout a little more.

1. Open the Week 3 pack:
   - [Week 03 // Dual Pathways](https://github.com/awayzo/add-open-courses/tree/main/languages/vbnet/cs120-command-nexus/weeks/week-03-dual-pathways)
2. Open the `templates` folder.
3. Choose the pathway you were assigned or selected:
   - `AgraRouteRecallTemplate`
   - or `AsteroidResponseAlertTemplate`
4. If you only want to inspect the files, you can read them directly in GitHub.
5. If you want the files on your computer, go back to the main repo page and use
   **Code -> Download ZIP**.

## How To Open The Project In Visual Studio

After downloading and extracting the files:

1. Open the correct template folder.
2. Find the `.vbproj` file.
3. Double-click it, or open it through Visual Studio.
4. Wait for the project to load.
5. Look at the **Error List** if the template is intentionally broken on
   purpose.

That last step matters. Some of these templates are designed to teach you how to
notice and fix startup issues instead of expecting every file to already be
perfect.
